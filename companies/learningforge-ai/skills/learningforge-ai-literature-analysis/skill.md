# Learning Forge AI - Literature Analysis Skill

## Overview
Scans academic and technical literature for construction domain knowledge. Identifies research trends and builds citation networks.

## When to Use
- When academic database scanning is needed
- When technical paper analysis is required
- When citation network building is needed
- When research trend identification is required
- **Don't use when:** General web research is sufficient

## Core Procedures

### Literature Analysis Workflow
1. **Database Scanning** - Search academic sources
2. **Paper Identification** - Select relevant papers
3. **Citation Analysis** - Build citation networks
4. **Trend Analysis** - Identify research directions
5. **Summary Generation** - Create literature summaries

## Sources
- Google Scholar
- ResearchGate
- Technical paper repositories
- Construction research journals
