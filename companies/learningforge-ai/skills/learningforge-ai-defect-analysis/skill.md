# Learning Forge AI - Defect Analysis Skill

## Overview
Analyzes defect patterns from internal logs and public case studies. Develops mitigation strategies and case study materials.

## When to Use
- When defect pattern analysis is needed
- When root cause analysis is required
- When mitigation strategy development is needed
- When case study synthesis is required
- **Don't use when:** General quality assessment is sufficient

## Core Procedures

### Defect Analysis Workflow
1. **Pattern Recognition** - Identify recurring defect patterns
2. **Root Cause Analysis** - Determine underlying causes
3. **Mitigation Development** - Create prevention strategies
4. **Case Study Synthesis** - Build learning materials
5. **Pattern Generalization** - Extend findings beyond single project

## Sources
- Internal defect logs
- Public defect databases
- Case study repositories
- Industry quality reports
