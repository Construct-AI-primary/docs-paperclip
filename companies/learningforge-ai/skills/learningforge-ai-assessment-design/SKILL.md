# Learning Forge AI - Assessment Design Skill

## Overview
Designs assessments and evaluation instruments for measuring construction professional competency. Creates mastery tests aligned with learning objectives.

## When to Use
- When assessment design is needed
- When evaluation instrument creation is required
- When mastery test development is needed
- When competency measurement tools are required

## Core Procedures

### Assessment Design Workflow
1. **Objective Mapping** - Map to learning objectives
2. **Question Design** - Create assessment questions
3. **Difficulty Calibration** - Calibrate question difficulty
4. **Validation** - Validate assessment effectiveness
5. **Feedback Integration** - Add constructive feedback
