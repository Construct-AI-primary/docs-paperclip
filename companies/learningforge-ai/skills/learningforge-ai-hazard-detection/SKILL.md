# Learning Forge AI - Hazard Detection Skill

## Overview
Detects construction hazards and develops hazard awareness training. Analyzes patterns to improve safety outcomes.

## When to Use
- When hazard detection is needed
- When hazard pattern analysis is required
- When safety awareness training is needed
- When risk identification training is required

## Core Procedures

### Hazard Detection Workflow
1. **Pattern Analysis** - Analyze hazard patterns
2. **Risk Identification** - Identify potential hazards
3. **Awareness Content** - Develop awareness training
4. **Prevention Training** - Create prevention content
5. **Update Integration** - Update with new hazards

**Primary Agent:** learningforge-ai-hazard-pattern-analyst
