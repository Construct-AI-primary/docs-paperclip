# Learning Forge AI - Feedback Synthesis Skill

## Overview
Aggregates feedback from multiple sources and creates synthesized feedback summaries. Identifies patterns across feedback for improvement.

## When to Use
- When feedback aggregation is needed
- When feedback synthesis is required
- When improvement recommendations are needed
- When feedback pattern analysis is required

## Core Procedures

### Feedback Synthesis Workflow
1. **Collection** - Gather feedback from multiple sources
2. **Categorization** - Organize feedback by topic
3. **Pattern Detection** - Identify common themes
4. **Synthesis** - Create consolidated summaries
5. **Recommendation Generation** - Generate improvement suggestions
