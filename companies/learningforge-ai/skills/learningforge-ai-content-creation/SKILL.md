# Learning Forge AI - Content Creation Skill

## Overview
Generates micro-lessons and educational content for construction domain training. Transforms knowledge into digestible learning modules.

## When to Use
- When micro-lesson generation is needed
- When learning content creation is required
- When educational material development is needed
- When training content production is required

## Core Procedures

### Content Creation Workflow
1. **Knowledge Extraction** - Extract key concepts from source material
2. **Chunk Design** - Create digestible content chunks
3. **Format Development** - Format for different learning styles
4. **Quality Review** - Validate content accuracy
5. **Delivery Preparation** - Prepare for learning platform

**Primary Agent:** learningforge-ai-micro-lesson-generator

Writes to `/workspace/lessons/content/`
