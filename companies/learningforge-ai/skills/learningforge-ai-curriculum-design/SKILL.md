# Learning Forge AI - Curriculum Design Skill

## Overview
Designs learning curricula and sequences learning paths for optimal knowledge acquisition. Creates progressive lesson plans based on prerequisite relationships.

## When to Use
- When curriculum design is needed
- When learning path sequencing is required
- When lesson plan development is needed
- When progressive learning structure design is required

## Core Procedures

### Curriculum Design Workflow
1. **Topic Analysis** - Analyze learning topics and dependencies
2. **Sequencing** - Determine optimal learning order
3. **Prerequisite Mapping** - Identify prerequisite relationships
4. **Milestone Creation** - Design learning milestones
5. **Assessment Integration** - Add knowledge checks

**Primary Agent:** learningforge-ai-curriculum-planner
