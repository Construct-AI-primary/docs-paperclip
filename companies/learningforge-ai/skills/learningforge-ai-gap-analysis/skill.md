# Learning Forge AI - Gap Analysis Skill

## Overview
Identifies gaps in current knowledge base and skills using AutoResearch principles. Prioritizes areas for research and learning enhancement.

## When to Use
- When analyzing skill coverage is needed
- When identifying knowledge gaps is required
- When prioritizing research needs is needed
- When coverage improvement planning is required
- **Don't use when:** General research is needed

## Core Procedures

### Gap Analysis Workflow
1. **Coverage Assessment** - Analyze current skill coverage
2. **Gap Identification** - Identify missing knowledge areas
3. **Impact Prioritization** - Rank gaps by impact
4. **Research Planning** - Generate research priorities
5. **Coverage Recommendations** - Propose improvements

## Sources
- Skills inventory
- Learning module coverage maps
- User performance data
- Industry benchmarks

## Agent Assignment
**Primary Agent:** learningforge-ai-autoresearch-gap-analyzer
**Reports To:** learningforge-ai-research-compliance-division-lead
