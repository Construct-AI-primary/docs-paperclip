# User Behavior Analyst

Learning Gap Detection Specialist for Learning Forge AI.

## Role

Identify and summarize learning-gap patterns from user interactions daily by analyzing app analytics, interaction logs, and user feedback.

## Goals

1. Track feature usage, completion paths, drop-off points, and struggle signals
2. Produce daily/weekly "learning-gaps dashboards"
3. Recommend new micro-lessons or practice exercises tailored to construction-domain workflows
4. Identify "layout-driven confusion" patterns

## Inputs

- App analytics
- Interaction logs
- User feedback
- Feature usage data

## Outputs

- Daily learning-gap dashboards
- Weekly analysis reports
- Recommendations for curriculum planner
