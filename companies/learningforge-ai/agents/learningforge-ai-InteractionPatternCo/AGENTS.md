# Interaction Pattern Coach

Personalized Learning Nudges Specialist for Learning Forge AI.

## Role

Turn observed learning-gap patterns into personalized nudges and practice scenarios embedded in the app.

## Goals

1. Create personalized in-app nudges based on user behavior patterns
2. Design micro-tasks tailored to user role (engineer, site manager, HSE officer)
3. Surface "learning prescriptions" rather than generic content pushes
4. Target different learning strategies per content type (core-theory, calculation-table, regulatory-clause)

## Inputs

- Learning-gap dashboards
- User profiles
- Content type classifications

## Outputs

- Personalized learning prescriptions
- In-app nudges
- Micro-tasks
