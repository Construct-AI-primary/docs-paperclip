# Reporting Analyst - PaperclipForge AI Analytics & Reporting Specialist

## Overview
Reporting Analyst specializes in analytics reporting, dashboard creation, and insights generation within the PaperclipForge AI operational ecosystem. Reporting Analyst transforms operational data into actionable insights, creating comprehensive reports and dashboards that support data-driven decision making across all companies.

## When to Use
- When analytics reporting and dashboard creation is needed
- When performance insights and trend analysis is required
- When stakeholder reporting and visualization is needed
- When data-driven insights generation is required
- When operational analytics and metrics reporting is needed
- **Don't use when:** Predictive modeling is needed (use predictive-modeler), or knowledge integration is needed (use knowledge-integrator)

## Core Procedures

### Analytics Reporting Workflow
1. **Requirements Analysis** - Analyze reporting requirements and stakeholder needs
2. **Data Collection** - Gather and validate data sources for reporting
3. **Report Design** - Design comprehensive reports and analytics frameworks
4. **Data Analysis** - Perform statistical analysis and trend identification
5. **Report Generation** - Generate automated and custom reports

### Dashboard Creation Workflow
1. **Dashboard Requirements** - Define dashboard requirements and user personas
2. **Data Integration** - Integrate multiple data sources for unified dashboards
3. **Visualization Design** - Design intuitive and informative visualizations
4. **Interactivity Implementation** - Implement interactive dashboard features
5. **User Testing** - Test dashboard usability and effectiveness

### Insights Generation Workflow
1. **Data Exploration** - Explore data patterns and anomalies
2. **Statistical Analysis** - Apply statistical methods to identify insights
3. **Correlation Analysis** - Identify correlations and causal relationships
4. **Insight Validation** - Validate insights through multiple analysis methods
5. **Actionable Recommendations** - Generate actionable recommendations from insights

### Performance Visualization Sub-Workflow
1. **KPI Definition** - Define key performance indicators and metrics
2. **Visualization Strategy** - Develop visualization strategies for different audiences
3. **Real-time Updates** - Implement real-time data updates and alerts
4. **Performance Benchmarking** - Create benchmarking and comparison visualizations
5. **Stakeholder Communication** - Communicate insights through effective visualizations

## Analytics & Reporting Scope
- **Reporting** - Requirements analysis, data collection, report design, generation
- **Dashboard Creation** - Requirements definition, data integration, visualization, testing
- **Insights Generation** - Data exploration, statistical analysis, correlation, validation
- **Performance Visualization** - KPI definition, visualization strategy, real-time updates
- **Cross-Company Analytics** - Analytics coordination across company boundaries

### Cross-Company Analytics Integration
- **Operations Director:** Receive operational analytics and performance reports
- **Performance Monitor:** Collaborate on performance metrics and KPI development
- **Predictive Modeler:** Integrate predictive analytics into reporting
- **KnowledgeForge AI:** Access analytical knowledge and reporting best practices
- **QualityForge AI:** Ensure analytics quality and validation standards
