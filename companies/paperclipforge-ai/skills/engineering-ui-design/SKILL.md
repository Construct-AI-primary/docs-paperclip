# Engineering UI Design Skill

## Overview

Specialized skill for creating engineering-specific user interfaces that address the unique requirements of technical workflows, complex data visualization, and domain-specific user experiences in engineering applications.

## Routing Criteria

### Primary Triggers
- Tasks requiring engineering visualization interfaces
- CAD/BIM system integration requirements
- Technical dashboard and control panel design
- Engineering workflow user experience optimization
- Complex data representation needs

### Secondary Triggers
- Scientific visualization requirements
- Technical specification review interfaces
- Engineering process monitoring dashboards
- Domain-specific user interface challenges

## Capabilities

### Core Competencies
- **Engineering UI/UX Design**: Creation of interfaces specifically for engineering workflows
- **Technical Visualization**: Complex data representation and visualization techniques
- **CAD/BIM Integration**: Interface design for computer-aided design systems
- **Dashboard Design**: Engineering process monitoring and control interfaces
- **Domain Adaptation**: User experience optimization for technical domains

### Technical Skills
- Interface prototyping and wireframing
- Data visualization design principles
- CAD system user experience patterns
- Engineering workflow optimization
- Technical user research and testing

## Examples

### FEA Result Visualization
- Stress/strain contour plots with interactive controls
- Result comparison interfaces with overlay capabilities
- Material property visualization dashboards
- Failure analysis result presentation

### CAD Model Interaction
- 3D model manipulation interfaces
- Parametric design control panels
- Assembly visualization and interaction
- Design review and markup interfaces

### Engineering Dashboards
- Process monitoring and control interfaces
- Quality assurance dashboard design
- Production tracking and analytics displays
- Engineering KPI visualization panels
