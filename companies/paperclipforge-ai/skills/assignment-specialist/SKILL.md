# Assignment Specialist - PaperclipForge AI Intelligent Routing Specialist

## Overview
Assignment Specialist specializes in intelligent agent routing, expertise matching, and workload balancing within the PaperclipForge AI operational ecosystem. Assignment Specialist analyzes issue requirements, matches them to agent expertise, and optimizes assignments for efficient resolution across all companies.

## When to Use
- When intelligent issue routing to appropriate agents is needed
- When expertise matching and skill assessment is required
- When workload balancing across teams is needed
- When assignment optimization and efficiency analysis is required
- When cross-company agent coordination is needed
- **Don't use when:** Issue creation is needed (use issue-generator), or progress tracking is needed (use progress-tracker)

## Core Procedures

### Intelligent Routing Workflow
1. **Issue Analysis** - Analyze issue requirements, complexity, and domain expertise needed
2. **Agent Matching** - Match issue requirements to agent skills and availability
3. **Expertise Assessment** - Assess agent expertise levels and success rates
4. **Workload Evaluation** - Evaluate current agent workloads and capacity
5. **Assignment Optimization** - Optimize assignments for efficiency and success probability

### Expertise Matching Workflow
1. **Skill Mapping** - Map issue requirements to required skills and competencies
2. **Agent Profiling** - Profile agent skills, experience, and performance history
3. **Compatibility Scoring** - Score agent-issue compatibility based on multiple factors
4. **Backup Identification** - Identify backup agents for critical assignments
5. **Assignment Validation** - Validate assignment quality and success probability

### Workload Balancing Workflow
1. **Capacity Monitoring** - Monitor agent capacity and current workload levels
2. **Load Distribution** - Distribute workload evenly across available agents
3. **Priority Management** - Manage assignment priorities based on urgency and importance
4. **Resource Optimization** - Optimize resource utilization across teams
5. **Performance Tracking** - Track assignment success rates and agent performance

### Cross-Company Coordination Sub-Workflow
1. **Inter-Company Routing** - Route issues across different company boundaries
2. **Collaboration Facilitation** - Facilitate collaboration between assigned agents
3. **Dependency Management** - Manage dependencies between cross-company assignments
4. **Communication Coordination** - Coordinate communication between assigned parties
5. **Success Monitoring** - Monitor success of cross-company assignments

## Assignment Management Scope
