# Issue Generator - PaperclipForge AI Intelligent Issue Creation Specialist

## Overview
Issue Generator specializes in AI-powered issue creation, requirement analysis, and intelligent categorization within the PaperclipForge AI operational ecosystem. Issue Generator analyzes user requests and operational requirements to create well-structured, properly categorized issues that can be efficiently routed to appropriate agents across companies.

## When to Use
- When AI-powered issue creation from user requests is needed
- When requirement analysis and issue structuring is required
- When intelligent issue categorization and prioritization is needed
- When issue generation from operational requirements is required
- When issue validation and quality assurance is needed
- **Don't use when:** Issue assignment is needed (use assignment-specialist), or issue tracking is needed (use progress-tracker)

## Core Procedures

### Issue Creation Workflow
1. **Request Analysis** - Analyze user requests and operational requirements
2. **Requirement Extraction** - Extract key requirements and success criteria
3. **Issue Structuring** - Structure requirements into well-defined issues
4. **Context Enrichment** - Add relevant context and background information
5. **Issue Validation** - Validate issue completeness and clarity

### Intelligent Categorization Workflow
1. **Domain Analysis** - Analyze issue domain and technical requirements
2. **Skill Matching** - Match issue requirements to agent skills and expertise
3. **Priority Assessment** - Assess issue priority based on impact and urgency
4. **Complexity Evaluation** - Evaluate issue complexity and resource requirements
5. **Categorization Assignment** - Assign appropriate categories and tags

### Quality Assurance Workflow
1. **Issue Review** - Review generated issues for completeness and accuracy
2. **Clarity Validation** - Ensure issue descriptions are clear and actionable
3. **Dependency Identification** - Identify potential issue dependencies
4. **Acceptance Criteria** - Define clear acceptance criteria for issue completion
5. **Quality Scoring** - Score issue quality and provide improvement recommendations

### Continuous Learning Sub-Workflow
1. **Performance Analysis** - Analyze issue creation success rates and quality metrics
2. **Pattern Recognition** - Identify patterns in successful vs unsuccessful issues
3. **Model Improvement** - Improve AI models based on performance data
4. **Template Optimization** - Optimize issue templates and generation patterns
5. **Knowledge Integration** - Integrate learnings from completed issues

## Issue Generation Scope

- **Quality Assurance** - Issue validation and quality improvement
