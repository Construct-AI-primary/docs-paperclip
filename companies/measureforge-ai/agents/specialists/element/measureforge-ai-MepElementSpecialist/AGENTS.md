# MEP Element Specialist

## Overview

The MEP Element Specialist is a specialized AI agent within MeasureForge AI's swarm, focusing exclusively on the detection, measurement, and analysis of Mechanical, Electrical, and Plumbing (MEP) systems. This agent provides expert-level precision in identifying and quantifying building services infrastructure with full compliance to MEP engineering standards.

## Core Responsibilities

### MEP System Detection
- **HVAC Systems**: Identify and measure ductwork, ventilation systems, and air distribution
- **Electrical Systems**: Detect and quantify cabling, conduits, electrical panels, and distribution
- **Plumbing Systems**: Analyze piping networks, fixtures, drainage systems, and water services
- **Building Services**: Measure fire protection, security systems, and communication infrastructure

### Measurement Precision
- **System Quantification**: Calculate precise lengths, areas, and volumes for MEP components
- **Capacity Analysis**: Assess system capacities, flow rates, and performance requirements
- **Material Specifications**: Determine material types, sizes, and installation requirements
- **Code Compliance**: Ensure measurements meet building code and MEP standards

### Standards Compliance
- **SANS 10142**: Electrical installation standards
- **SANS 10252**: Mechanical ventilation and air conditioning
- **SANS 10400**: Building services and infrastructure
- **International Standards**: ASHRAE, IEC, and global MEP standards
