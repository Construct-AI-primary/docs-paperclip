# MeasureForge AI Shared Resources

This directory contains shared resources, utilities, and frameworks used across all MeasureForge AI agents and systems.

## Directory Structure

```
shared/
├── cad-integration-framework.md    # Unified CAD system integration
├── standards-framework.md          # Multi-jurisdictional standards support
├── measurement-algorithms.md       # Core measurement calculation algorithms
├── visual-overlay-system.md        # Audit trail and visualization framework
├── swarm-coordination.md           # Multi-agent orchestration protocols
├── procurement-integration.md      # Procurement workflow integration
└── quality-assurance.md            # Measurement validation and QA frameworks
```

## Core Frameworks

### CAD Integration Framework
Unified interface for all CAD system integrations:
- AutoCAD, Revit, Civil 3D, MicroStation, SketchUp, ArchiCAD, SolidWorks
- Standardized data models and transformation protocols
- Real-time bidirectional synchronization
- Error handling and recovery mechanisms

### Standards Framework
Multi-jurisdictional compliance automation:
- ASAQS (South African Quantity Surveying Standards)
- SANS 1200 MJ (South African National Standards)
- International standards (BS, ASTM, ISO)
- Dynamic standards updates and version management

### Measurement Algorithms
Core calculation engines:
- Geometric measurement algorithms
- Quantity take-off automation
- Standards-compliant calculations
- Error detection and correction

### Visual Overlay System
Audit trail and visualization:
- Non-destructive CAD overlays
- Real-time compliance visualization
- Multi-user collaboration overlays
- Historical change tracking

### Swarm Coordination
Multi-agent orchestration:
- Task distribution algorithms
- Conflict resolution protocols
- Performance optimization
- Quality assurance monitoring

### Procurement Integration
End-to-end procurement pipeline:
- 01900 procurement workflow integration
- Contract tender compilation
- Supplier integration protocols
- Cost optimization algorithms

### Quality Assurance
Measurement validation framework:
- Automated accuracy checking
- Standards compliance validation
- Error pattern recognition
- Continuous improvement algorithms

## Usage Guidelines

### For Agent Developers
1. **Import shared frameworks** at the top of agent implementations
2. **Extend base classes** for specialized functionality
3. **Use standardized interfaces** for CAD and procurement integrations
4. **Follow naming conventions** for shared resources

### For System Integrators
1. **Reference framework documentation** for integration patterns
2. **Use provided utilities** for common operations
3. **Extend frameworks** for new CAD systems or standards
4. **Contribute improvements** back to shared resources

### For Quality Assurance
1. **Use QA frameworks** for automated testing
2. **Reference standards** for compliance validation
3. **Monitor performance** using shared metrics
4. **Report issues** through established channels
