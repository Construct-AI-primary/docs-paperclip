# MeasureForge AI Organizational Chart

## Overview

This document outlines the organizational structure for MeasureForge AI's procurement operations, specifically focused on the builder procurement segregation workflows.

## Executive Leadership

**Directors** (Top Level - Report to CEO/Executive Team)
- All directors report to the MeasureForge AI CEO or executive leadership
- Directors oversee major functional areas and strategic initiatives

## Procurement Operations Structure

### Directors

1. **📏 Measurement Director** (`measureforge-ai-measurement-director`)
   - **Role**: Oversees measurement and classification operations
   - **Responsibilities**: CAD integration, measurement standards, procurement classification
   - **Reports To**: CEO/Executive Leadership

2. **🛒 Procurement Director** (`measureforge-ai-procurement-director`)
   - **Role**: Oversees all procurement operations
   - **Responsibilities**: Supplier management, contract administration, procurement compliance
   - **Reports To**: CEO/Executive Leadership

3. **🔗 Integration Director** (`measureforge-ai-integration-director`)
   - **Role**: Oversees system integrations and APIs
   - **Responsibilities**: API integration, data flow supervision, integration standards
   - **Reports To**: CEO/Executive Leadership

4. **💰 Finance Director** (`measureforge-ai-finance-director`)
   - **Role**: Oversees budget and financial operations
   - **Responsibilities**: Financial planning, cost control, financial reporting
   - **Reports To**: CEO/Executive Leadership

5. **⚙️ Operations Director** (`measureforge-ai-operations-director`)
   - **Role**: Oversees operational workflows and approvals
   - **Responsibilities**: Workflow governance, approval processes, operational compliance
   - **Reports To**: CEO/Executive Leadership

### Specialist Agents

#### Measurement Director's Team

- **🧙 Sage Weaver** - Procurement Classifier
  - Analyzes CAD measurements and classifies them into procurement categories
  - Expert in material type recognition and trade specialization
  - **Reports To**: Measurement Director

#### Procurement Director's Team

- **🔨 Forge Master** - Subcontract Procurement Agent
  - Handles specialized trade subcontracts (electrical, plumbing, HVAC, fire protection)
  - Expert in RFQ generation, subcontractor evaluation, CIDB verification
  - **Reports To**: Procurement Director

- **👑 Resource Baron** - Materials Procurement Agent
  - Manages purchase orders for materials and equipment
  - Expert in Candy system integration and bulk ordering optimization
  - **Reports To**: Procurement Director

- **🧠 Service Sage** - Service Procurement Agent
  - Coordinates testing, maintenance, and inspection services
  - Expert in service requirement analysis and provider certification
  - **Reports To**: Procurement Director

- **💪 Task Titan** - Work Order Procurement Agent
  - Manages internal labor and construction task management
  - Expert in work order scope definition and labor resource allocation
  - **Reports To**: Procurement Director

- **💕 Match Maker** - Supplier Matching Agent
  - Matches procurement requirements against approved supplier lists
  - Expert in supplier capability analysis and compliance scoring (B-BBEE, ISO 9001)
  - **Reports To**: Procurement Director

#### Integration Director's Team

- **🍬 Candy Connector** - Candy Integration Agent
  - Direct export to Candy procurement system
  - Expert in Candy API integration and material category mapping
  - **Reports To**: Integration Director

#### Finance Director's Team

- **🛡️ Budget Guardian** - Budget Monitoring Agent
  - Real-time budget tracking and variance monitoring
  - Expert in budget tracking, variance analysis, and cost optimization
  - **Reports To**: Finance Director

#### Operations Director's Team

- **🚪 Gate Keeper** - Approval Routing Agent
  - Integration with PROC-ORDER approval workflows
  - Expert in approval matrix configuration and escalation handling
  - **Reports To**: Operations Director

## Functional Areas

### Procurement Categories Handled

1. **Subcontracts** (Forge Master)
   - Electrical, plumbing, HVAC, fire protection, structural steel
   - RFQ generation and subcontractor evaluation
   - CIDB and B-BBEE compliance verification

2. **Materials & Equipment** (Resource Baron)
   - Purchase orders for construction materials
   - Candy system integration
   - Bulk ordering and supplier catalog management

3. **Services** (Service Sage)
   - Testing, maintenance, inspection services
   - Provider certification verification
   - Service order generation and timeline scheduling

4. **Internal Labor** (Task Titan)
   - Work order management
   - Labor resource allocation
   - Equipment scheduling and productivity tracking

5. **Supplier Matching** (Match Maker)
   - Supplier capability analysis
   - Geographic coverage mapping
   - Compliance scoring and B-BBEE verification

### Support Functions

6. **System Integration** (Candy Connector)
   - Candy procurement system integration
   - API connectivity and data flow
   - Order submission and status tracking

7. **Financial Oversight** (Budget Guardian)
   - Real-time budget monitoring
   - Variance analysis and alerting
   - Cost optimization recommendations

8. **Approval Workflows** (Gate Keeper)
   - PROC-ORDER system integration
   - Approval matrix configuration
   - Escalation handling and notifications

9. **Classification** (Sage Weaver)
   - CAD measurement interpretation
   - Procurement category classification
   - Material type recognition
