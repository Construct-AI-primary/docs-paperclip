# Architectural Design Procurement Typology

## Sector Overview
Architectural design procurement covers the full spectrum of design services from concept through construction documentation, including building information modeling (BIM), 3D visualization, rendering, and peer review. Design procurement involves specialized workstations and software, visualization hardware, and professional design consultancy services across multiple design stages.

## Common PO Types (Purchase Orders for Goods)

| PO Category | Typical Goods | Key Specifications | Lead Times |
|-------------|---------------|-------------------|------------|
| Workstations | High-performance desktops, laptops, render nodes | NVIDIA Quadro/RTX, 64GB+ RAM, multi-core processors | 2-4 weeks |
| Software Licenses | BIM (Revit, ArchiCAD), CAD (AutoCAD), rendering (V-Ray, Lumion) | Perpetual or subscription, network licensing, multi-seat bundles | 1-4 weeks |
| BIM Hardware | VR headsets, 3D scanners, drones, laser levels | FARO/Leica accuracy specs, BIM-compatible export formats | 2-8 weeks |
| Plotting Equipment | Large-format plotters, scanners, color printers | 36" / 44" width, archival ink, high DPI scanning | 2-4 weeks |
| Visualization Equipment | Render nodes, GPU arrays, VR systems | CUDA cores, VRAM capacity, VR-ready specifications | 3-6 weeks |
| Model Making Supplies | 3D printers, laser cutters, modeling materials | Layer resolution, build volume, material compatibility | 2-6 weeks |

## Common SO Tracking Requirements

| Category | SO Focus Areas | Critical SO Verification Points |
|----------|---------------|--------------------------------|
| Software Procurement | License delivery, activation, seat allocation | License key receipt, user assignment confirmation |
| BIM Hardware | Equipment calibration, software compatibility | Calibration certificates, BIM software compatibility test |
| Render Farm | GPU delivery, installation, cooling setup | Render node benchmarking, network integration test |
| Plotting Consumables | Ink/toner supplies, paper stock, maintenance kits | Media profile availability, color calibration targets |

## Common WO Types (Work Orders for Services)

| WO Category | Typical Services | Compliance Requirements |
|-------------|-----------------|------------------------|
| Design Consultancy | Concept design, schematic design, design development | Professional indemnity insurance, registration with SACAP |
| BIM Modeling | 3D modeling, clash detection, quantity take-offs | BIM execution plan (BEP), LOD specification compliance |
| Visualization | Photorealistic rendering, VR walkthroughs, animations | Render quality standards, delivery format specifications |
| Peer Review | Design review, code compliance check, constructability review | SAIA scope definitions, reviewer independence requirement |
| Sustainable Design | Energy modeling, daylight analysis, green star certification | SANS 10400 compliance, GBCSA accreditation |
| Heritage Impact | Heritage impact assessments, conservation management plans | NHRA compliance, heritage authority approvals |

## Key Goods vs Services Distinctions for Architectural Design

### Goods (PO Items)
- High-performance workstations with specific GPU and memory configurations for BIM/rendering
- Software licenses with subscription management and seat allocation tracking
- VR/visualization hardware with BIM software compatibility testing
- Large-format plotters with archival ink and color calibration requirements

### Services (WO/Service Order Items)
- Design consultancy with staged deliverables and milestone reviews
- BIM modeling with defined LOD requirements and CDE protocols
- Visualization services with defined output formats and resolution standards
- Peer review with independence requirements and professional registration

## Project-Specific Procurement Considerations
- **RIBA Stages:** Stage-specific deliverables and procurement gates (Stage 1-7)
- **BIM Level 2:** Information management per ISO 19650, CDE setup, common data environment protocols
- **Visualization:** Render quality tiers (draft, medium, photorealistic), VR integration with design reviews
- **Sustainable Design:** Green Star, LEED, EDGE certification requirements influence consultant scope
- **Model Making:** Physical models for planning applications, competition entries, public exhibitions
- **Design Competition:** Honorarium agreements, intellectual property rights, exhibition and publication rights
