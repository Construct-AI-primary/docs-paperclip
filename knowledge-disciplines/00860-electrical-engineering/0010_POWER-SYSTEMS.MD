# Power Systems Procurement Typology

## Sector Overview
Power systems procurement covers power generation, transmission and distribution infrastructure, electrical protection and control systems, and renewable energy integration. Procurement involves high-value electrical equipment with strict IEC/SANS compliance, type testing, and grid code requirements.

## Common PO Types (Purchase Orders for Goods)

| PO Category | Typical Goods | Key Specifications | Lead Times |
|-------------|---------------|-------------------|------------|
| Transformers | Power transformers, distribution transformers, earthing transformers | IEC 60076, SANS 780, ONAN/ONAF cooling, impedance spec | 20-50 weeks |
| Switchgear | MV/HV SF6/air-insulated switchgear, ring main units | IEC 62271, SANS 10086, short-circuit rating, arc fault tested | 16-35 weeks |
| Protection Relays | Numerical relays, Feeder/Busbar/Transformer protection | IEC 61850, IEC 60255, communication protocols | 8-16 weeks |
| Cables | HV/MV/LV power cables, control cables, fibre optic | SANS 1507/1429, XLPE/PVC, fire-rated, armour type | 8-20 weeks |
| Batteries & UPS | Lead-acid/Li-ion batteries, UPS, chargers | IEC 60896, IEEE 1184, autonomy time, temperature spec | 8-16 weeks |
| Metering | Energy meters, CTs/VTs, power quality meters | IEC 62053, revenue metering accuracy, meter test cert | 6-12 weeks |

## Common SO Tracking Requirements

| Category | SO Focus Areas | Critical SO Verification Points |
|----------|---------------|--------------------------------|
| Transformers | Core/coil manufacture, tank assembly, testing | Type test certs, routine test report, tap changer verification |
| Switchgear | Panel fabrication, busbar assembly, HV testing | Dielectric test, contact resistance, relay integration test |
| Cables | Conductor production, insulation, sheathing | HV test certificate, drum length verification, bend test |
| Protection Systems | Relay configuration, logic testing, SCADA integration | Functional test, communication protocol test, IED configuration |

## Common WO Types (Work Orders for Services)

| WO Category | Typical Services | Compliance Requirements |
|-------------|-----------------|------------------------|
| System Design | Load flow, short-circuit, protection coordination, arc flash | IEC 60909, IEC 60255, IEEE 1584, SANS 10142 |
| Protection Coordination | Grading study, relay setting calculation, logic diagram | Protection philosophy document, setting report, test sheet |
| Commissioning | HV testing, relay commissioning, SCADA integration | Commissioning spec, SAT, energisation sequence, test reports |
| Testing Services | Insulation resistance, PI, tan delta, sweep frequency response | IEEE 43, IEC 60076 test procedures |
| Maintenance Services | Oil sampling, thermography, partial discharge, circuit breaker timing | OEM maintenance schedule, maintenance management system |
| Earthing & Lightning | Earthing design, lightning risk assessment, surge protection | SANS 10199, SANS 62305, touch/step potential calc |

## Key Goods vs Services Distinctions for Power Systems

### Goods (PO Items)
- Transformers with type test certificates and factory acceptance testing
- Switchgear with short-circuit rating certification and arc fault testing
- Protection relays with IEC 61850 compliance and communication protocol configuration
- Cables with HV test certificates and fire-performance classification

### Services (WO/Service Order Items)
- System design with protection coordination and arc flash hazard analysis
- Commissioning with site acceptance testing and energisation procedure
- Maintenance services with OEM-approved procedures and test equipment certification
- Earthing design with touch/step potential compliance to SANS 10199

## Project-Specific Procurement Considerations
- **Generation:** Generator step-up transformers, unit auxiliary transformers, excitation systems
- **Transmission:** HV circuit breakers, disconnect switches, CVTs, line traps, series compensation
- **Distribution:** Ring main units, pad-mounted transformers, LV switchboards, metering panels
- **Substations:** GIS (gas insulated switchgear), AIS (air insulated switchgear), control building integration
- **Renewable Integration:** Inverters, BESS, PCC metering, grid code compliance testing
- **Industrial:** MCCs, VSDs, soft starters, heating/cooling controls, hazardous area electrical equipment
