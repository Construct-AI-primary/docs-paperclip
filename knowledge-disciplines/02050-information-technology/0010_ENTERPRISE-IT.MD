# Enterprise IT Project Procurement Typology

## Sector Overview
Enterprise IT projects encompass hardware procurement, software licensing, cloud services deployment, network infrastructure, and cybersecurity implementations. Procurement involves both capital equipment (hardware) and operational services (SaaS, cloud), with significant licensing and compliance requirements.

## Common PO Types (Purchase Orders for Goods)

| PO Category | Typical Goods | Key Specifications | Lead Times |
|-------------|---------------|-------------------|------------|
| Server Hardware | Rack servers, blade systems, storage arrays, backup appliances | OEM specs, CPU cores, RAM capacity, disk IOPS, redundant PSU | 4-16 weeks |
| Networking Equipment | Switches, routers, firewalls, load balancers, APs | 10/25/40/100GbE, PoE, throughput, port density | 4-12 weeks |
| Endpoint Devices | Laptops, desktops, monitors, tablets, mobile devices | CPU gen, RAM, SSD, security features, warranty | 2-8 weeks |
| Software Licenses | OS, database, office productivity, development tools | Licensing model (per-user/per-core), version, support term | 1-4 weeks |
| Cybersecurity Appliances | Firewalls, IDS/IPS, SIEM appliances, encryption devices | Throughput, IPS throughput, concurrent connections, certifications | 6-16 weeks |
| Structured Cabling | CAT6/6A/8, fiber optic OM3/OM4/OS2, patch panels, trays | TIA/EIA standards, fire rating, insertion loss, length | 2-6 weeks |

## Common SO Tracking Requirements

| Category | SO Focus Areas | Critical SO Verification Points |
|----------|---------------|--------------------------------|
| Server/Storage | Manufacturing, configuration, burn-in | OEM configuration validation, burn-in test results, warranty activation |
| Networking Equipment | Configuration, firmware version | VLAN config, firmware version, support contract activation |
| Software Licenses | License key delivery, entitlement verification | License count, subscription end date, support tier |
| Cloud Subscriptions | Tenant provisioning, resource quotas | Region selection, SLA tier, budget alerts |

## Common WO Types (Work Orders for Services)

| WO Category | Typical Services | Compliance Requirements |
|-------------|-----------------|------------------------|
| Implementation | Server installation, network deployment, system migration | OEM procedures, security standards, change management |
| Managed Services | 24/7 monitoring, helpdesk, patch management, backup management | SLA commitments, response times, escalation procedures |
| Cloud Migration | Workload migration, re-platforming, cloud-native adoption | Security assessment, data sovereignty, migration plan |
| Cybersecurity | Penetration testing, SOC services, incident response, audits | ISO 27001, PCI DSS, NIST, industry standards |
| Consulting | Architecture design, technology roadmapping, vendor selection | Professional certifications, methodology standards |
| Training | End-user training, administrator training, certification prep | Training materials, competency assessment, certification exams |

## Key Goods vs Services Distinctions for Enterprise IT

### Goods (PO Items)
- Hardware with OEM warranty and maintenance requirements
- Software with license compliance, true-up, and audit requirements
- Networking equipment with configuration and testing requirements
- Security appliances with certification and compliance requirements

### Services (WO/Service Order Items)
- Implementation with project methodology and change control requirements
- Managed services with SLA commitments and escalation procedures
- Cloud migration with security, compliance, and cutover planning
- Cybersecurity assessments with regulatory compliance frameworks

## Project-Specific Procurement Considerations
- **Data Center Build-out:**
Power and cooling requirements, rack layout, structured cabling standards, fire suppression
- **Cloud Migration:**
Data transfer costs, bandwidth requirements, downtime windows, application compatibility
- **Cybersecurity Program:**
Risk assessment, regulatory compliance (POPIA/GDPR), incident response planning, security awareness training
- **Digital Workplace:**
Device standardization, MDM/UEM deployment, collaboration tools, identity management
- **Business Continuity:**
DR site selection, backup infrastructure, replication bandwidth, failover testing procedures
