# Security Systems Procurement Typology

## Sector Overview
Security procurement covers access control systems, video surveillance, perimeter security, intrusion detection, cybersecurity, and integrated security management for commercial, industrial, and critical infrastructure projects. Procurement involves electronic security equipment with IP networking integration, compliance with security standards, and specialist design and monitoring services.

## Common PO Types (Purchase Orders for Goods)

| PO Category | Typical Goods | Key Specifications | Lead Times |
|-------------|---------------|-------------------|------------|
| Cameras | IP cameras, PTZ, thermal, ANPR, body cameras | ONVIF profile G/T/S, resolution (2MP-12MP), IR range | 4-10 weeks |
| Access Control | Card readers, controllers, door locks, biometrics | TCP/IP communication, Wiegand/OSDP protocol, credential type | 4-10 weeks |
| Barriers & Gates | Boom barriers, turnstiles, bollards, vehicle gates | Duty cycle, opening speed, crash rating (IWA 14-1) | 6-14 weeks |
| Alarm Systems | Intrusion detectors, glass break, PIR, beam detectors | EN 50131, detection zone, false alarm immunity | 2-6 weeks |
| Cybersecurity | Firewalls, endpoint protection, SIEM, IDS/IPS | NIST CSF, ISO 27001, firewall throughput, UTM features | 2-8 weeks |
| Monitoring Station | CCTV workstation, video management server, storage | RAID configuration, retention period, operator interface | 4-8 weeks |

## Common SO Tracking Requirements

| Category | SO Focus Areas | Critical SO Verification Points |
|----------|---------------|--------------------------------|
| Camera Supply | Delivery, firmware, lens test | Image quality test, ONVIF compliance test, firmware version |
| Access Control | Panel configuration, credential programming, comms test | Reader test, credential issue, door lock power test |
| Alarm System | Panel programming, sensor placement, communication | Zone setup, communication path test, alarm verification |
| Servers & Storage | Hardware delivery, RAID setup, VMS installation | Storage capacity check, failover test, backup verification |

## Common WO Types (Work Orders for Services)

| WO Category | Typical Services | Compliance Requirements |
|-------------|-----------------|------------------------|
| Security Assessments | Risk assessment, threat analysis, security audit | ISO 31000, security survey, vulnerability assessment |
| System Design | Security concept, design specification, equipment schedule | SANS 10400-S, Electronic security standards, coverage analysis |
| Installation | Cabling, mounting, configuration, labelling | SANS 10142, installation standard, cable test, termination |
| Integration | VMS integration, BMS integration, PSIM | API/SDK compatibility, system commissioning, integration test |
| Monitoring Services | Remote monitoring, alarm response, patrol | PSIRA registration, response time SLA, incident reporting |
| Maintenance | Preventive maintenance, repair, firmware updates | OEM maintenance schedule, SLA response time, spare parts holding |

## Key Goods vs Services Distinctions for Security Systems

### Goods (PO Items)
- IP cameras with ONVIF profile compliance and resolution specification
- Access control hardware with TCP/IP communication and OSDP protocol support
- Barriers and gates with crash rating certification and duty cycle verification
- Cybersecurity hardware with throughput specification and UTM feature compliance

### Services (WO/Service Order Items)
- Security assessments with risk methodology and vulnerability analysis
- System design with coverage analysis and security standard compliance
- Installation with SANS 10142 cabling standard and cable certification
- Monitoring services with PSIRA registration and response time SLA

## Project-Specific Procurement Considerations
- **Critical Infrastructure:** NIPP compliance, dual-path communication, physical and logical security integration
- **Perimeter Security:** Zone-based detection, layered approach, CCTV coverage overlap, lighting coordination
- **Integration:** VMS-BMS-ACS integration, single-event single-operator response, PSIM deployment
- **Cybersecurity:** Network segmentation, secure remote access, firmware patch management, SOC integration
- **Retrofit:** Existing system compatibility, phased deployment, legacy equipment migration
- **POPIA/GDPR:** Privacy impact assessment, data retention policy, subject access request handling
