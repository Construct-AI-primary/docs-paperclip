# Emergency Response Project Procurement Typology

## Sector Overview
Emergency response procurement covers fire detection and suppression systems, emergency lighting, evacuation equipment, spill response materials, and emergency response planning services. Procurement must comply with NFPA, local fire codes, and emergency response planning standards.

## Common PO Types (Purchase Orders for Goods)

| PO Category | Typical Goods | Key Specifications | Lead Times |
|-------------|---------------|-------------------|------------|
| Fire Detection | Smoke/heat detectors, alarm panels, pull stations, notification devices | UL/NFPA listed, detection type (ionization/photo), zone capacity | 4-10 weeks |
| Fire Suppression | Sprinkler heads, valves, piping, FM-200/Novec, foam systems | NFPA 13/NFPA 2001, K-factor, coverage area, discharge rate | 8-20 weeks |
| Emergency Lighting | Exit signs, emergency luminaires, central battery systems | Lumens, runtime (90 min), IP rating, photoluminescent | 2-6 weeks |
| Evacuation Equipment | Evac chairs, stair descent devices, assembly point signs | Weight capacity, descent speed, storage dimensions, signage | 2-6 weeks |
| Spill Response Kits | Spill kits, absorbent booms, containment booms, PPE | Volume capacity, chemical compatibility, disposal method | 1-4 weeks |

## Common SO Tracking Requirements

| Category | SO Focus Areas | Critical SO Verification Points |
|----------|---------------|--------------------------------|
| Fire Detection | Factory configuration, device compatibility | Addressing plan, zone assignment, system compatibility |
| Fire Suppression | Pipe schedule, valve spec compliance | Hydrostatic test certs, valve pressure ratings |
| Emergency Lighting | Battery type, runtime compliance | Battery chemistry, runtime test results |

## Common WO Types (Work Orders for Services)

| WO Category | Typical Services | Compliance Requirements |
|-------------|-----------------|------------------------|
| Emergency Planning | EAP development, drill coordination, evacuation planning | Regulatory compliance, site-specific plans, drill records |
| Hazmat Response | Spill response, hazmat cleanup, waste disposal | HAZWOPER, DOT/ADR regulations, waste manifests |
| Fire Warden Training | Warden appointment, training, drill facilitation | Accreditation, training records, competency assessment |
| Maintenance/Inspection | Fire system testing, extinguisher inspection, emergency light testing, alarm test | NFPA 72/25, monthly/annual test schedules, log books |

## Key Goods vs Services Distinctions for Emergency Response

### Goods (PO Items)
- Fire detection with addressing, zoning, and integration requirements
- Suppression systems with code compliance and hydrostatic testing
- Emergency lighting with battery runtime and photoluminescent certification

### Services (WO/Service Order Items)
- Emergency planning with site-specific customization and regulatory compliance
- Hazmat response with certified personnel and waste tracking
- Fire warden training with accredited coursework and practical drills

## Project-Specific Procurement Considerations
- **High-Occupancy Buildings:** Enhanced alarm systems, voice evacuation, stair pressurization
- **Industrial/Hazmat Facilities:** Foam suppression, gas detection, hazmat storage areas
- **Healthcare Facilities:** Code blue integration, area of refuge, evacuation chair management
- **High-Rise Buildings:** Stairwell pressurization, phased evacuation, communication systems
- **Remote/Offshore:** Self-sufficient emergency response equipment, satellite communication, medevac planning
