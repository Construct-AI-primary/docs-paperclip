# Geotechnical Procurement Typology

## Sector Overview
Geotechnical procurement covers site investigation, soil and rock testing, ground improvement, instrumentation, and foundation design services for civil and structural projects. Procurement involves drilling equipment, sampling tools, laboratory testing apparatus, and geotechnical instrumentation with strict testing standards compliance.

## Common PO Types (Purchase Orders for Goods)

| PO Category | Typical Goods | Key Specifications | Lead Times |
|-------------|---------------|-------------------|------------|
| Drilling Equipment | Drill rigs, augers, core barrels, casing | SANS 634, diamond bit size, HQ/NQ/PQ core spec | 4-12 weeks |
| Sampling Tools | Shelby tubes, split spoon samplers, piston samplers | ASTM D1587 (fixed piston), D1586 (SPT), sample diameter | 1-4 weeks |
| Testing Apparatus | Triaxial, direct shear, consolidation, CBR apparatus | ASTM D2850/D3080, SANS 3001, calibration cert | 4-10 weeks |
| Instrumentation | Piezometers, inclinometers, settlement markers, strain gauges | Geosense/Roctest spec, vibrating wire, automatic datalogging | 4-8 weeks |
| Consumables | Core boxes, sample bags, drilling mud, additives | Sample retention requirements, bentonite/PHPA spec | 1-2 weeks |
| Field Testing Kit | DCP, shear vane, permeability apparatus, nuclear densometer | ASTM D6951, D2573, calibration cert, radiation license | 2-6 weeks |

## Common SO Tracking Requirements

| Category | SO Focus Areas | Critical SO Verification Points |
|----------|---------------|--------------------------------|
| Drilling Programme | Borehole progress, core recovery, sampling | Core recovery % (target > 95%), SPT N-values, RQD logging |
| Laboratory Testing | Test scheduling, specimen preparation, reporting | Chain of custody, test method compliance, hold time adherence |
| Instrumentation | Data logger setup, baseline readings, communication | Baseline readings within 24hrs, telemetry confirmation |
| Consumable Supply | Stock levels, delivery coordination | Sample box availability, drilling fluid stock, bit inventory |

## Common WO Types (Work Orders for Services)

| WO Category | Typical Services | Compliance Requirements |
|-------------|-----------------|------------------------|
| Borehole Drilling | Rotary, percussion, sonic drilling; core logging | SANS 634, hole depth/angle spec, geotechnical logging standards |
| CPT Testing | Cone penetration test, pore pressure, seismic | ASTM D5778, 10-tonne capacity, cone calibration |
| Laboratory Testing | Classification, strength, consolidation, chemical | SANAS accreditation, ASTM/BS/ISO methods, hold times |
| Slope Stability Analysis | Limit equilibrium, FEM, hazard assessment | Eurocode 7 / SANS 10160, accepted FoS criteria |
| Ground Improvement | Vibro-compaction, grouting, soil mixing, wick drains | Geotechnical design spec, settlement criteria, QA/QC testing |
| Instrumentation Monitoring | Data collection, interpretation, reporting | Monitoring frequency, alarm thresholds, automated reporting |

## Key Goods vs Services Distinctions for Geotechnical

### Goods (PO Items)
- Drilling equipment with appropriate capacity, tooling, and depth capability
- Sampling tools with ASTM standard compliance and sample integrity requirements
- Testing apparatus with SANAS/ISO calibration and method compliance
- Geotechnical instrumentation with automated datalogging and telemetry

### Services (WO/Service Order Items)
- Borehole drilling with core recovery targets and geotechnical logging
- CPT testing with calibrated cone and pore pressure measurement
- Laboratory testing with accredited methods and hold time compliance
- Ground improvement with performance verification testing

## Project-Specific Procurement Considerations
- **Foundation Design:** Bearing capacity assessment, settlement analysis, foundation type recommendation
- **Slope Stability:** Geotechnical model development, failure mechanism analysis, remediation design
- **Groundwater Control:** Dewatering design, permeability testing, groundwater monitoring plan
- **Earthworks Support:** Material classification, compaction specification, CBR testing
- **Rock Engineering:** Rock mass classification, joint orientation analysis, support design
- **Seismic Hazard:** Seismic refraction, MASW, borehole geophysics, site classification per building code
