# Process Plant Procurement Typology

## Sector Overview
Process plant procurement covers reactors, columns, heat exchangers, piping systems, instrumentation, and control systems for chemical, petrochemical, pharmaceutical, and industrial process facilities. Procurement involves pressure vessels, rotating equipment, and precision instrumentation with strict material certification, ASME/ASTM compliance, and hazardous area requirements.

## Common PO Types (Purchase Orders for Goods)

| PO Category | Typical Goods | Key Specifications | Lead Times |
|-------------|---------------|-------------------|------------|
| Pressure Vessels | Reactors, columns, separators, storage tanks | ASME VIII div 1/2, PED, NACE MR0175, PWHT requirements | 20-50 weeks |
| Pumps | Centrifugal, positive displacement, diaphragm, metering | API 610/674, ANSI B73.1, NPSHr, material class | 12-30 weeks |
| Compressors | Centrifugal, reciprocating, screw, rotary | API 617/619, capacity control, vibration spec | 30-60 weeks |
| Control Valves | Globe, ball, butterfly, relief, control valves | IEC 60534, Fisher/Masonelian spec, SIL rating | 10-20 weeks |
| Piping & Fittings | Carbon steel, stainless, alloy piping, flanges, fittings | ASME B31.3, ASTM A106/A312, wall schedule, material cert | 6-16 weeks |
| Instrumentation | Pressure, temperature, flow, level transmitters; analyzers | ATEX/IECEx, SIL rating, 4-20mA HART/Ff protocol | 8-16 weeks |

## Common SO Tracking Requirements

| Category | SO Focus Areas | Critical SO Verification Points |
|----------|---------------|--------------------------------|
| Vessels & Columns | Material procurement, rolling, welding, NDE | Material certs, weld map, RT/UT reports, hydrostatic test |
| Rotating Equipment | Component machining, assembly, performance test | Witnessed performance test, vibration analysis, seal test |
| Valve Packages | Body casting/machining, seat testing, actuation | Shell test, seat leakage test, fire-safe cert (API 607) |
| Instrument Loops | Sensor calibration, loop check, functional test | Calibration cert, loop check documentation, SIL proof test |

## Common WO Types (Work Orders for Services)

| WO Category | Typical Services | Compliance Requirements |
|-------------|-----------------|------------------------|
| Process Design | PFD, P&ID development, process simulation (Aspen/Hysys) | ISO 9001 design procedures, design basis document |
| Piping Design | Pipe routing, stress analysis, isometric drawing | ASME B31.3, Caesar II stress analysis, support design |
| Hazard Analysis | HAZOP, LOPA, SIL assessment, QRA | IEC 61882, IEC 61508/61511, facility siting study |
| Equipment Installation | Lifting, alignment, piping hook-up, bolt torquing | Installation procedure, OEM supervision, torque records |
| Commissioning | Pre-commissioning, loop check, operational test | Commissioning manual, punch list sign-off, performance test |
| Integrity Management | RBI assessment, NDT inspection, corrosion monitoring | API 580/581, ASME PCC-1, corrosion loop management |

## Key Goods vs Services Distinctions for Process Plant

### Goods (PO Items)
- Pressure vessels with ASME U-stamp, material traceability, and NDE certification
- Rotating equipment with witnessed performance testing and mechanical run tests
- Control valves with SILL assessment and fire-safe testing for hydrocarbon service
- Instrumentation with hazardous area certification and calibration traceability

### Services (WO/Service Order Items)
- Process design with HAZOP support and design basis documentation
- Piping design with stress analysis and materials selection
- Hazard analysis with formal HAZOP/LOPA facilitation and close-out
- Commissioning with performance guarantee testing against process guarantees

## Project-Specific Procurement Considerations
- **Corrosive Service:** Material selection (duplex, super duplex, hastelloy), corrosion allowance, PWHT requirements
- **High Pressure:** API/ASME code requirements, flanged vs welded connections, hydrotest procedures
- **High Temperature:** Creep consideration, alloy selection (chrome-moly), expansion joint design
- **Hazardous Area:** ATEX/IECEx zoning, equipment certification, earthing/bonding requirements
- **Clean Service:** Stainless steel passivation, orbital welding, cleanliness spec for oxygen/pharma
- **Brownfield Tie-ins:** Shutdown coordination, hot tap procedures, isolation and blinding requirements
