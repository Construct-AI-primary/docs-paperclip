# Voice Communication Project Procurement Typology

## Sector Overview
Voice communication procurement covers VoIP phones, headsets, audio equipment, conferencing hardware, PBX systems, and professional telephony implementation services. This shared discipline supports communication infrastructure across all project types.

## Common PO Types

| PO Category | Typical Goods | Key Specifications | Lead Times |
|-------------|---------------|-------------------|------------|
| VoIP Phones | SIP desk phones, conference phones, DECT cordless | PoE support, codec support, display type, line keys | 2-4 weeks |
| Headsets | Wired/wireless headsets, noise-canceling, DECT headsets | Connectivity (USB/BT/DECT), noise canceling, battery life | 1-2 weeks |
| Audio Equipment | Speakerphones, microphone arrays, amplifiers, soundbars | Coverage area, microphone pickup, echo cancellation | 2-6 weeks |
| Conferencing Hardware | Video bars, room kits, PTZ cameras, touch panels | Camera resolution, FOV, AI features, platform compatibility | 2-6 weeks |
| PBX Systems | On-premise PBX, SBC, gateway, session management | Concurrent calls, trunk capacity, codec transcoding | 4-12 weeks |

## Common WO Types

| WO Category | Typical Services | Compliance Requirements |
|-------------|-----------------|------------------------|
| Telephony Implementation | PBX installation, SIP trunking, phone provisioning, DECT setup | Network readiness, QoS configuration, testing plan |
| Call Center Setup | ACD configuration, IVR setup, call recording, reporting | Industry standards, compliance recording, agent management |
| Integration | CRM integration, UCaaS integration, directory sync | API specs, user mapping, data synchronization |
| Managed Voice Services | 24/7 monitoring, fault management, moves/adds/changes | SLA commitments, response times, escalation procedures |

## Key Distinctions
### Goods (PO Items)
- VoIP phones with network compatibility and PoE requirements
- Audio equipment with room sizing and acoustic coverage
- PBX hardware with capacity planning and redundancy

### Services (WO Items)
- Implementation with QoS configuration and acceptance testing
- Integration with API specifications and user provisioning
- Managed services with SLA commitments and fault management

## Project-Specific Considerations
- **Enterprise Deployment:** Centralized dial plan, multi-site networking, emergency call routing
- **Contact Center:** Automatic call distribution (ACD), IVR scripting, call recording, workforce management
- **Remote Workforce:** Remote phone provisioning, VPN/STUN connectivity, home office equipment
- **Multi-Site:** QOS across WAN, site failover, centralized management, inter-site dialing
