# Workflows Common Discipline

## Overview

The **workflows-common** discipline provides universal workflow templates and conversion procedures that apply across multiple other disciplines in the Paperclip ecosystem.

## Purpose

- **Template Standardization**: Create consistent workflow templates across disciplines
- **Cross-Discipline Support**: Enable workflow sharing between different disciplines
- **Procedure Development**: Build reusable procedures for common tasks
- **Automation Support**: Facilitate automation through standardized triggers
