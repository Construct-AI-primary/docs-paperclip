# AGENTS.md - Safety Hazard Workflow Project

## Agent Instructions

This file contains project-specific instructions for agents working on the Safety Hazard workflow.

## Project Agents

### Primary Agents

- **domainforge-ai-architectural-integration**: Hazard assessment coordination
- **safety-domainforge**: Hazard management automation

## Workflow Tasks

### Issue Assignment

Issues are assigned based on:
1. Platform (desktop/mobile/web)
2. Hazard severity level
3. Mitigation priority

### Status Management

All issues follow the standard workflow:
- `backlog` → `todo` → `in_progress` → `in_review` → `done`
