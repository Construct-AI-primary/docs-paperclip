# AGENTS.md - Architectural Workflow Project

## Agent Instructions

This file contains project-specific instructions for agents working on the Architectural workflow.

## Project Agents

### Primary Agents

- **domainforge-ai-architectural-integration**: Architectural design coordination
- **integrateforge-ai-ada-autocad**: CAD integration for architectural documentation

### Supporting Agents

- **design-integration-domainforge**: Design workflow integration

## Workflow Tasks

### Issue Assignment

Issues are assigned based on:
1. Platform (desktop/mobile/web)
2. Building type and complexity
3. Code compliance requirements

### Status Management

All issues follow the standard workflow:
- `backlog` → `todo` → `in_progress` → `in_review` → `done`
