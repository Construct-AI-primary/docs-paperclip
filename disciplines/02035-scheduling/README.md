# 02035 - Scheduling

The Scheduling discipline covers project scheduling, timeline management, resource leveling, and critical path analysis for construction and engineering projects.

## Overview

This discipline ensures project schedules are developed, maintained, and monitored effectively to meet delivery targets and optimize resource utilization.

## Key Responsibilities

- Schedule development and sequencing
- Critical path method analysis
- Resource leveling and optimization
- Schedule monitoring and reporting
- Delay analysis and mitigation
- Milestone tracking
- Schedule risk assessment
- Integration with project controls
