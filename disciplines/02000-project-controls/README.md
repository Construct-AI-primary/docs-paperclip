# 02000 - Project Controls

The Project Controls discipline covers scheduling, cost control, progress measurement, earned value management, and project performance monitoring.

## Overview

This discipline ensures projects are tracked effectively, costs are controlled, schedules are maintained, and performance metrics are accurately measured and reported.

## Key Responsibilities

- Project scheduling and sequencing
- Cost estimating and control
- Progress measurement and reporting
- Earned value management
- Risk monitoring
- Change control management
- Performance analytics
- Project forecasting
