# DESIGN-WORKFLOW Project

## Overview

The DESIGN-WORKFLOW project defines the workflow templates for the Design Management discipline (00800). It covers design coordination, interdisciplinary review, design quality management, and design programme management across all engineering and architectural disciplines.
