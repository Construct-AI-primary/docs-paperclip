# 00890 - Projects Director

The Projects Director discipline covers project governance, delivery oversight, portfolio management, and strategic alignment of all projects within the organization.

## Overview

This discipline ensures projects are delivered on time, within budget, and to quality standards while maintaining alignment with organizational strategy and stakeholder expectations.

## Key Responsibilities

- Portfolio governance and oversight
- Project delivery assurance
- Resource allocation optimization
- Risk and issue management
- Stakeholder communication
- Performance monitoring
- Project methodology enforcement
- Continuous improvement initiatives
