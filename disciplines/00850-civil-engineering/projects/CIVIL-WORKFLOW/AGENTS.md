# AGENTS.md - Civil Engineering Workflow Project

## Agent Instructions

This file contains project-specific instructions for agents working on the Civil Engineering workflow.

## Project Agents

### Primary Agents

- **domainforge-ai-architectural-integration**: Civil design coordination
- **integrateforge-ai-ada-autocad**: CAD integration for civil documentation
- **structural-integration-domainforge**: Structural engineering coordination

### Supporting Agents

- **design-integration-domainforge**: Design workflow integration

## Workflow Tasks

### Issue Assignment

Issues are assigned based on:
1. Platform (desktop/mobile/web)
2. Project phase and complexity
3. Site conditions and constraints

### Status Management

All issues follow the standard workflow:
- `backlog` → `todo` → `in_progress` → `in_review` → `done`

### Cross-Discipline Coordination

Civil workflows require coordination with:
- 00800-design
- 00855-geotechnical-engineering
- 00860-electrical-engineering
