# 02075 - Inspection

The Inspection discipline covers quality inspection, testing, verification, and conformance assessment for construction and engineering works.

## Overview

This discipline ensures works meet specified requirements, inspections are conducted properly, testing protocols are followed, and conformance certificates are issued appropriately.

## Key Responsibilities

- Inspection planning and scheduling
- Quality inspection execution
- Testing and sampling
- Conformance verification
- Non-conformance reporting
- Inspection documentation
- Third-party inspection coordination
- Statutory inspection compliance
