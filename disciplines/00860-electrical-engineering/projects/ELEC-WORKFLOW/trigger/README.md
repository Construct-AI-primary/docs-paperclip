# ELEC-WORKFLOW Triggers

This directory contains automation triggers for the ELEC-WORKFLOW project.

## Available Triggers

- **ELEC-WORKFLOW-000**: Project automation trigger (to be created)

## Trigger Configuration

Triggers are defined per the `discipline_workflow_conversion_procedure.md` and activate when:
- A new electrical engineering project is created
- A workflow template is assigned to an agent
- A workflow template status changes
