# AGENTS.md - Electrical Engineering Workflow Project

## Agent Instructions

This file contains project-specific instructions for agents working on the Electrical Engineering workflow.

## Project Agents

### Primary Agents

- **domainforge-ai-architectural-integration**: Electrical design coordination
- **integrateforge-ai-ada-autocad**: CAD integration for electrical documentation
- **electrical-integration-domainforge**: Electrical systems coordination

### Supporting Agents

- **design-integration-domainforge**: Design workflow integration

## Workflow Tasks

### Issue Assignment

Issues are assigned based on:
1. Platform (desktop/mobile/web)
2. Voltage level and complexity
3. Safety criticality

### Status Management

All issues follow the standard workflow:
- `backlog` → `todo` → `in_progress` → `in_review` → `done`

### Cross-Discipline Coordination

Electrical workflows require coordination with:
- 00800-design
- 00850-civil-engineering
- 00870-mechanical-engineering
