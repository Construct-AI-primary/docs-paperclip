# AGENTS.md - PROC-BUDGET Budget Monitoring Workflow

## Agent Instructions

This file contains project-specific instructions for agents working on the PROC-BUDGET budget monitoring and cost control workflow.

## Project Agents

### Primary Agents

- **interface-devforge**: UI/UX development for budget dashboards
- **codesmith-devforge**: Core functionality and budget calculations
- **devcore-devforge**: Backend integration and API development

### Supporting Agents

- **database-infraforge**: Database and infrastructure
- **guardian-qualityforge**: Testing and QA
- **validator-qualityforge**: Validation
- **doc-analyzer-knowledgeforge**: Documentation

## Workflow Tasks

### Issue Assignment

Issues are assigned based on:
1. Platform (desktop/mobile/web)
2. Budget workflow priority
3. Financial criticality

### Status Management

All issues follow the standard workflow:
- `backlog` → `todo` → `in_progress` → `in_review` → `done`

### Cross-Discipline Coordination

Budget workflows require coordination with:
- 01900-procurement
- Financial systems
- Project management
- Cost control systems

## Budget Workflow Codes

| Code | Workflow | Priority |
|------|----------|----------|
| BUD-PLAN | Budget Planning & Allocation | High |
| BUD-MON | Spend Monitoring & Tracking | High |
| BUD-VAR | Variance Analysis & Reporting | High |
| BUD-CTL | Cost Reduction Initiatives | Medium-High |
| BUD-REA | Budget Reallocation & Approval | Medium |
