# PROC-AUDIT Shared Data Models Module

## Overview

This module contains shared data models and TypeScript interfaces used across all platform implementations (desktop, mobile, web).

## Data Models

### Audit Models
- `audit-event.model.ts` - Audit event structure
- `audit-log.model.ts` - Audit log entry
- `audit-query.model.ts` - Query parameters

### Compliance Models
- `compliance-rule.model.ts` - Compliance rule definition
- `compliance-violation.model.ts` - Violation record
- `compliance-score.model.ts` - Compliance score

### Control Models
- `control.model.ts` - Control definition
- `control-test.model.ts` - Test results
- `control-effectiveness.model.ts` - Effectiveness metrics

### Report Models
- `report.model.ts` - Report definition
- `report-submission.model.ts` - Submission record
- `report-archive.model.ts` - Archive metadata
