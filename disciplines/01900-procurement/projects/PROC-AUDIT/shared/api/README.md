# PROC-AUDIT Shared API Module

## Overview

This module contains shared API contracts and interfaces used across all platform implementations (desktop, mobile, web).

## API Components

### Audit API
- `audit-capture.ts` - Event capture endpoint
- `audit-query.ts` - Query and retrieval endpoints
- `audit-export.ts` - Export functionality

### Compliance API
- `compliance-evaluate.ts` - Rule evaluation endpoints
- `compliance-violations.ts` - Violation management
- `compliance-reports.ts` - Compliance reporting

### Control API
- `control-test.ts` - Control testing endpoints
- `control-effectiveness.ts` - Effectiveness metrics
- `control-remediation.ts` - Remediation tracking

### Reporting API
- `report-generate.ts` - Report generation
- `report-submit.ts` - Report submission
- `report-archive.ts` - Report archival
