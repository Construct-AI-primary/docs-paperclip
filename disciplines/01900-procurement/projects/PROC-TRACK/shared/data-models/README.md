# PROC-TRACK Shared Data Models

## Overview

This directory contains shared data models used across desktop, mobile, and web platforms.

## Data Models

### Order Tracking Models
- `OrderTracking`: Core tracking entity
- `Milestone`: Delivery milestone entity
- `Delay`: Delay information entity

### Expediting Models
- `ExpeditingRequest`: Expediting request entity
- `Escalation`: Escalation entity

### Communication Models
- `Notification`: Notification entity
- `CustomerMessage`: Customer message entity

## Usage

These shared models should be used by all platform implementations to ensure consistent data structures.
