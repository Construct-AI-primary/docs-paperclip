# PROC-TRACK Shared API Contracts

## Overview

This directory contains shared API contracts used across desktop, mobile, and web platforms.

## API Modules

### Tracking API
- Order tracking endpoints
- Milestone monitoring endpoints
- Delay detection endpoints

### Expediting API
- Expediting request endpoints
- Supplier engagement endpoints
- Escalation endpoints

### Communication API
- Notification endpoints
- Customer messaging endpoints
- Alert endpoints

## Usage

Platform implementations should import and extend these shared contracts for consistency.
