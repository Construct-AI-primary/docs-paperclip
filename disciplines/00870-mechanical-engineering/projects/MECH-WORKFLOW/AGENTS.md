# AGENTS.md - Mechanical Engineering Workflow Project

## Agent Instructions

This file contains project-specific instructions for agents working on the Mechanical Engineering workflow.

## Project Agents

### Primary Agents

- **domainforge-ai-architectural-integration**: Mechanical design coordination
- **integrateforge-ai-ada-autocad**: CAD integration for mechanical documentation
- **mechanical-integration-domainforge**: HVAC and equipment coordination

### Supporting Agents

- **design-integration-domainforge**: Design workflow integration

## Workflow Tasks

### Issue Assignment

Issues are assigned based on:
1. Platform (desktop/mobile/web)
2. System complexity
3. Energy efficiency requirements

### Status Management

All issues follow the standard workflow:
- `backlog` → `todo` → `in_progress` → `in_review` → `done`
