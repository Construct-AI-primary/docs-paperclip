# 00900 - Document Control

The Document Control discipline covers document management, version control, records management, and information governance for construction and engineering projects.

## Overview

This discipline ensures documents are properly created, reviewed, approved, distributed, and archived in accordance with organizational standards and regulatory requirements.

## Key Responsibilities

- Document creation and formatting standards
- Version control and revision management
- Document distribution and transmittal
- Records management and archiving
- Compliance with document control standards
- Quality management system documentation
- Electronic document management
- Audit trail maintenance
