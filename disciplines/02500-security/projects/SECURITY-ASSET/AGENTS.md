## Agent Instructions

This file contains project-specific instructions for agents working on the Security Asset workflow.

## Project Agents

### Primary Agents

- **domainforge-ai-architectural-integration**: Security coordination
- **security-domainforge**: Security asset tracking automation
- **integrateforge-ai-ada-autocad**: Security system documentation

## Workflow Tasks

### Issue Assignment

Issues are assigned based on:
1. Platform (desktop/mobile/web)
2. Security criticality
3. Asset value and risk level

### Status Management

All issues follow the standard workflow:
- `backlog` → `todo` → `in_progress` → `in_review` → `done`

### Cross-Discipline Coordination

Security workflows require coordination with:
- 00800-design
- 00850-civil-engineering
- 02400-safety
