# 02500 - Security

The Security discipline covers physical security, cybersecurity, asset protection, personnel security, and security incident management for construction and engineering projects.

## Overview

This discipline ensures physical and information assets are protected, security threats are identified and mitigated, security protocols are followed, and security incidents are managed effectively.

## Key Responsibilities

- Physical security management
- Cybersecurity and information protection
- Asset protection
- Personnel security vetting
- Security incident management
- Access control
- Security risk assessment
- Security training and awareness
