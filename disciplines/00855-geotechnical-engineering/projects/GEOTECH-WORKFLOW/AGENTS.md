# AGENTS.md - Geotechnical Engineering Workflow Project

## Agent Instructions

This file contains project-specific instructions for agents working on the Geotechnical Engineering workflow.

## Project Agents

### Primary Agents

- **domainforge-ai-architectural-integration**: Geotechnical coordination
- **integrateforge-ai-ada-autocad**: CAD integration for geotechnical documentation

### Supporting Agents

## Workflow Tasks

### Issue Assignment

Issues are assigned based on:
1. Platform (desktop/mobile/web)
2. Site investigation status
3. Foundation type requirements

### Status Management

All issues follow the standard workflow:
- `backlog` → `todo` → `in_progress` → `in_review` → `done`
